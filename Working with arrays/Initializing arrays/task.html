Initializing 2D Arrays

Like their one-dimensional counterparts, two-dimensional arrays can be initialized to default values or, if values are known in advance, an array initializer can be written into the source code.

Expressions Involving 2D Arrays

Assume nums has been initialized using one of the two constructors.

  • nums refers to the entire 2D array, itself
  • nums.length refers to the length of the outer array (number of rows)
  • nums[0].length refers to the length of an inner array (number of columns for row 0)
  • nums[i] refers to the i-th row in the array
  • there is no mechanism for referring to the i-th column in the array

Jagged/Ragged Arrays

It is possible to create an array of arrays where each row has a different number of columns. The resulting array is referred to as a "jagged" or "ragged" array. More information can be found at this Wikipedia article or at GeeksForGeeks
