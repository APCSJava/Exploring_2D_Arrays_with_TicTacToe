Dimensioning Arrays

Write code for the constructor such that each of the four fields is initialized as described.

The four fields rely on the constructor parameters as follows:

  • the field wide uses the smaller value as the number of rows and the larger value as the number of columns
  • the field tall uses the larger value as the number of rows and the smaller value as the number of columns
  • the field large uses the larger value as the number of both rows and columns
  • the field small uses the smaller value as the number of both rows and columns
Determine which of the two parameters, a or b, is smaller and which is larger.
An array can be initialized as follows: new type[numRows][numColumns];
The element type is different for each of the fields -- three hold arrays of arrays of primitives while one holds an array of arrays of reference type String.
