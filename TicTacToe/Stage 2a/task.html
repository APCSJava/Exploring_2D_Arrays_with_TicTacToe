Prepare for Printing

Override the toString method to generate a string suitable for printing to the console.

The desired output looks like the following:

[0, 0, 0]
[0, 0, 0]
[0, 0, 0]

Each row of the array can be rendered as a string using the toString method provided by the java.util.Arrays class.

Remember not to print to the console inside toString. Instead, focus on building and returning a string that would be a suitable representation of the object's data.

Use the toString method of the java.util.Arrays class to print each row in the array.
After each row, include a newline character (\n) so that the remainder of the string will be placed on a new line.
There are newline characters following the 1st and 2nd row but no newline character following the third row.
