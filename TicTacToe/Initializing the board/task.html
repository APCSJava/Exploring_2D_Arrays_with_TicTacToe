Notice the datatype associated with the board. We'll use integer types for this game. An empty square will be represented by 0 while squares taken by player 1 and 2 will be assigned values 1 and 2, respectively.

Inside the constructor, initialize the field board to an array of the appropriate type having three rows and three columns.

As the number of columns and number of rows are always 3, you can hard code that value into your initialization code.
Be sure to initialize the field 'board' and not a local variable of the same name.
