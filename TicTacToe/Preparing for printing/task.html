Implement the toString method to generate a descriptive string suitable for printing to the console.

The desired string should have three lines, one for each row of the board. Shown together, the three rows of output should look like the following:

[0, 0, 0]
[0, 0, 0]
[0, 0, 0]

Tip: The java.util.Arrays class provides a toString method which accepts an array as a parameter and produces the needed string for each row. Follow the link to learn more and then use it in your solution.

CAUTION! Do not print to the console. Instead, build and return a string that when printed (by other code) would produce the output shown.

Use the toString method of the java.util.Arrays class to print each row in the array.
After each row, include a newline character (\n) to start a fresh line at that point in the string. The newline character sequence is, itself, a string, so it must be placed within quotes in your source code.
There are newline characters following the 1st and 2nd row but no newline character following the third row.
